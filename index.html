A História do Html

(clique aqui para voltar)

Em 1980 o Fisico Britâncio Tim Berners-Lee iniciou um projeto baseado no conceito de hipertexto denominado Enquire, desenvolvido inicialmente em linguagem Pascal. Anos mais tarde, Tim Berners-Lee e Robert Cailliau conseguiram implementar a primeira comunicação entre um cliente e um servidor através da internet, o protocolo HTTP. Em 1989, surgia então a World Wide Web (www). Em conjunto à implementação da World Wide Web, Tim Berners-Lee lançou a linguagem HTML. A versão inicial do HTML foi baseada na SGML, uma linguagem de estruturação de documentos.

O que são Clients?

(clique aqui para voltar)

Os clients são basicamente computadores com Browser(navegadores), que são chamados de Clients sides. Esses navegadores monstram para os clients de forma interativa com uma interface elegante de como preencherem os formulários, como exemplos e enviarem para os servidores com suas informações importantes.

O que são Servers?

(clique aqui para voltar)

Os Servers são computadores especificos para servir os clientes e armazenar os dados que os clients enviam e os servidores mandam um response(resposta), para o cliente. Existem o banco de dados que armazenam essas informações importantes, que podem ser atualizadas, apagadas, escritas, lidas pelos servidores e programadas pelos devs backend.

Como é uma estrutura básica de um HTML?

(clique aqui para voltar)

O Html que é (HyperText Markup Language) é contém uma estrutura básica com a linguagem de marcação dos textos, que servem para marcar os textos com as Tags que são importantes para os motores de busca sejam: google, yahoo... que servem para redenrizar de forma correta as páginas do seu Site. A estrutura básica do Html é a seguinte: html, o head, o title, o body. Essas são importantes para estruturar o html. Usa-se no Body(corpo) as seguintes Tags(marcações) para editar seus textos: p: que significa parágrafo. h1 até h6: que é o tamanho do texto... entre a tag como exemplo input: que serve para o cliente adicionar dados, entre outros atributos como por exemplos: negrito, itálico e sublinhado.

Entendendo a diferença entre listas ordenadas e não ordenadas

(clique aqui para voltar)

As listas ordenadas as: ol que servem para criar listas numeradas e seu interior a: li que servem para adicionar o texto dentro dessa tags. Já as listas não ordenadas as: ul que servem para criar listas com marcações não numeradas. Como exemplos: de bolas pretas,quadradas... que servem como tópicos em um texto. Em seu interior a tag: li servem para adicionar o texto, em seu interior.
